1. Put this folder in your MAMP/WAMP or something similar

2. Create a local database called anything

3. Go to inc/db_credentials.php and change the parameters to match your database

4. visit http://localhost:8888/tutorial-project/api/install.php to add the table
to your database

5. go to articles.php and test it out (it just adds a person and gets all the persons
using angular and php)
